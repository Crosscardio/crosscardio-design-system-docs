05 / PADRÕES

Estados obrigatórios

Toda jornada operacional deve comunicar progresso, ausência de dados, falhas, confirmação e sucesso.

Loading

Skeleton preserva a estrutura durante o carregamento.

Empty

Explique o vazio e ofereça uma próxima ação.

Error

Descreva o problema e permita tentar novamente.

Confirm

Confirme ações destrutivas com contexto curto.

Success

Confirme o resultado e permita desfazer quando aplicável.

06 / QUALIDADE

Acessibilidade por padrão

Os componentes priorizam contraste, foco, semântica e áreas de interação adequadas.

✓ Contraste mínimo WCAG AA.
✓ Alvos de 44px web e 48px mobile.
✓ Estados de foco sempre visíveis.
✓ Erros possuem mensagem textual.
✓ Loading e disabled são semânticos.
✓ Navegação funciona por teclado.
