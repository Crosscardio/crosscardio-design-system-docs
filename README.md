# CrossCardio Design System — Documentação pública

Site estático público da documentação do CrossCardio Design System.

- Não contém o código-fonte dos pacotes privados.
- Não contém segredos, tokens de acesso ou dependências internas.
- As cores e padrões seguem o Figma atual do CrossCardio.

## Publicação

Ative uma única vez em **Settings → Pages**:

- Source: **Deploy from a branch**
- Branch: **main**
- Folder: **/ (root)**

A documentação ficará disponível em:

https://crosscardio.github.io/crosscardio-design-system-docs/
